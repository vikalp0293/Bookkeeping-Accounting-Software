QuickBooks Web Connector Configuration File
============================================

This folder contains the QuickBooks Web Connector configuration file (.qwc) 
needed to connect Sync Accounting Desktop with QuickBooks Desktop.

File: sync_accounting.qwc

SETUP INSTRUCTIONS:
==================

1. Open QuickBooks Web Connector (if not installed, download from Intuit)

2. In QuickBooks Web Connector, click "Add an application"

3. Browse to this folder and select "sync_accounting.qwc"

4. Enter the following credentials:
   - Username: admin
   - Password: [Your password - contact administrator]

5. Click "OK" to authorize the connection

6. Verify the connection status shows "Connected" in QuickBooks Web Connector

LOCATION:
=========

After installation, this file is located at:
[Installation Directory]\resources\sync_accounting.qwc

You can also access it from the application menu:
Tools → Open QuickBooks Web Connector Config

For more information, see the User Guide.
