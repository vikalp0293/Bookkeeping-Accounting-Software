Sync Accounting – QuickBooks Sync App

How to install and use the app on your Windows PC. Written for everyday users.

Getting started

This guide explains how to install and run the Sync Accounting app so it can sync data with QuickBooks Desktop. Follow the steps below in order. If your company gave you a different set of instructions, follow those instead.

What you need (download these 3 programs)

Before you start, you need to install three things on your computer. You can download them here, or your IT person may give you the files. Install them in this order: Python first, then QuickBooks SDK, then the Sync Accounting app.

Step 1: Install the three programs (in order)

  1. First: Download and install Python for Windows (row 1 above). Run the installer, follow the on-screen steps, then finish.
  2. Second: Download and install QuickBooks SDK (row 2 above). Run the installer and complete the setup.
  3. Third: Download and install the Sync Accounting app (row 3 above). Run the installer, click Next when asked, then Finish when done.
  4. When all three are installed, open the Sync Accounting app from the Start menu (look for Sync Accounting QB SDK).

Step 2: First time you open the app

The first time you open the app, it will check your computer. It may tell you that something is missing (for example Python or QuickBooks). If so:

  • Use the Download buttons in the table above to get the other two programs (QuickBooks SDK and Python), then run those installers too.
  • If the app shows an Install Dependencies or similar button, click it and wait for it to finish.

When everything is ready, you'll see a message like "All requirements are met! You can start syncing." Then you can move on to the next step.

Step 3: Sign in to your account

When you open the Sync Accounting app, you'll see a Sign in to your account screen.

  1. Enter your Email (the one your company gave you for Sync Accounting).
  2. Enter your Password.
  3. Click Sign in.

Step 4: Install and open QuickBooks (if you use it)

  1. If you don't have QuickBooks Desktop on your computer, install it (version 2018 or newer). Your company may provide it.
  2. Open QuickBooks Desktop.
  3. Open your usual company file (the one you use every day) and leave QuickBooks open. The sync app needs QuickBooks to be open with your company file when you sync.

Two accounts that must exist in QuickBooks

Before syncing, make sure your QuickBooks company file has these two accounts. If they don't exist, create them in QuickBooks first:

  • Miscellaneous Expense – used by the app to sync checks / withdrawals.
  • Sales – used by the app to sync deposits.

Step 5: Set up Settings (file path and folder)

In the Sync Accounting app, click Settings in the left menu. You need to fill in a few details so the app can connect to QuickBooks and know where your files are.

QuickBooks company file path

In the QuickBooks SDK Sync section:

  • QuickBooks company file path (.QBW) – This is the location of your QuickBooks company file on your computer (for example C:\Users\YourName\Documents\Company.qbw).
  • Open QuickBooks and your company file first. Then either:
    • Click Select folder to pick the folder where your company file is, then type the file name (e.g. Royal Ginger Inc - Athens.qbw) after the path, or
    • Type the full path to the file manually.
  • When you're done, click Save SDK settings.

Folder to monitor and QuickBooks account name

In the Desktop Settings section:

  • Folder to Monitor – Choose the folder where you will place PDF files for the app to process (e.g. bank statements). Click Browse... to select the folder.
  • QuickBooks Account Name – Enter the QuickBooks account name where all transactions will be synced. This account must already exist in QuickBooks (for example indore-bank).
  • If you want the app to start monitoring this folder automatically when you open the app, check Automatically start monitoring when app launches.
  • Click Save Settings.

Step 6: Start syncing

  1. Make sure QuickBooks is still open with your company file.
  2. In the Sync Accounting app, click Start sync.
  3. The app will connect to QuickBooks and sync your data. Wait until it finishes.

If you see an error message (for example "QuickBooks must be running" or "Company file not found"), check that QuickBooks is open with your company file and that you entered the company file path and other details correctly in Settings. If it still doesn't work, contact your IT support.

Step 7: Stopping and syncing again later

  • To stop syncing: click Stop sync in the app.
  • To sync again another day: open QuickBooks and your company file first, then open the Sync Accounting app and click Start sync again.

Checklist – do you have everything?

  • Windows 10 or 11 computer
  • QuickBooks Desktop (2018 or newer) installed
  • QuickBooks open with your company file when you sync
  • In QuickBooks: Miscellaneous Expense account (for checks/withdrawals) and Sales account (for deposits)
  • All three programs installed in order: Python, then QuickBooks SDK, then Sync Accounting app
  • Signed in to the app with your email and password
  • Settings filled in: QuickBooks company file path (.QBW), Folder to Monitor, QuickBooks Account Name, and Save SDK settings / Save Settings clicked
  • Internet connection
