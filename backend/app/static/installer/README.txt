INSTALLER DIRECTORY
===================

Place the Sync Accounting Desktop installer executable file here.

Recommended file name: SyncAccountingDesktop-Setup.exe

The installer will be accessible at:
https://dev-sync-api.kylientlabs.com/docs/download/installer

Supported file names (will be auto-detected):
- SyncAccountingDesktop-Setup.exe
- SyncAccountingDesktop-Setup-x64.exe
- sync-accounting-desktop-setup.exe
- installer.exe
